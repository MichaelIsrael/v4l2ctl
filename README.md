# v4l2ctl
v4l2ctl is a python package to control v4l2 drivers.

# Status
The project is still in its early development stages. It can, however, already
be used to read the name, driver, version and capabilities of a v4l2 device
driver.

# Install
```
pip install v4l2ctl
```
